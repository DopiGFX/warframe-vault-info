# warframe-vault-info
📁 Get Warframe's Prime Vault information


## How to run
1. Install Deno https://docs.deno.com/runtime/getting_started/installation/
2. Open project and install dependencies
   ```bash
   deno i
   ```
3. Running the script
  - Running once: 
    ```bash
    deno run fetch
    ```
  - Running on file change: 
    ```bash
    deno run dev
    ```
